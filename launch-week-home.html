The Product
60 min
Introducing Agent ObservabilityTrack and analyze the performance, behavior, and interactions of AI agents. Export to Honeycomb traces from agent sessions, explicit user feedback, eval results, and use Agent Timeline, BubbleUp, Canvas, Boards, and SLOs to understand what your agents are doing. Get insights into token usage, cost of major LLM models, model performance (bias, hallucination and drift).
Introducing Canvas Spatial workspaceAgents and Humans can work together to perform AI-native investigation using a multiplayer spatial workspace, a chat interface, dynamic UI elements and reusable skills.
Introducing Canvas Agent with skillsAn autonomous agent starts investigating when triggers fire or SLOs burn, before an engineer opens their laptop using Skills (reusable investigation playbooks) that encode expert knowledge and are applied automatically.
